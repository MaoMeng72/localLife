本地生活电商网站――
（1）首页：三级菜单-原生js实现；
	轮播图-自己封装的jq插件，可以自动轮播、点击按钮轮播、点击列表轮播；
	注销-localStora本地存储，储存了一个data，通过改变里面的字段名实现注销功能；
	购物车-未登录状态下不能点击购物车，自动跳转到登陆界面；
	楼梯+商品渲染-模块化，商品渲染用ajax拿json文件中的数据，判断商品具体顶部的距离，将距离传给floor.js，用jq时间楼梯功能。

（2）登陆注册：用了localstorage本地存储实现。

（3）购物车：当用户登录之后才能登录进去购物车，没有登录就直接跳转到登录界面。

（4）商品列表：点击商品列表会出现对应的商品详情。

（5）商品详情：实现了放大镜、tab、添加购物车的功能。只有一个商品商品详情页面，在列表或者首页点击商品会在本地存储商品的id,在详情页获取商品id拿到商品数据渲染页面。
